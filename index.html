MVP Scope

Core features (v1)
  • Item capture (camera/upload) with OCR → auto‑category + metadata (make/model/serial).
  • AI‑assisted value estimate (replacement‑cost range, beta).
  • Policy upload (PDF or photos) → basic gap highlights by category.
  • Inventory export (CSV/PDF) for claims & records.
For carriers & agents
  • Read‑only carrier dashboard (demo data) with adoption and top under‑insured categories.
  • Demo‑only coverage upgrade flow (no live binding or payments).
  • Pilot access via invite codes; consent tracking.
Privacy & controls
  • Clear consent toggle + “delete my data” request flow.
  • Secure storage & limited access for the pilot environment.
Not in this MVP
  • Direct carrier integrations or real policy changes.
  • In‑app payments or endorsements binding.
  • Native mobile app (web app only for now).
  • Open API.
Pilot: closed beta; rolling invites; target cohort under 1,000 participants across carriers, agents, and policyholders. Carriers can request a pilot slot via the waitlist.
